# Admin Dashboard

Internship project: A React + Bootstrap admin dashboard with a collapsible sidebar, dynamic tabs, and tab-based customer information forms integrated with APIs.

## Features
- Collapsible sidebar with nested submenus
- Dynamic, route-aware tabs
- Customer information forms (Personal, Address, Introducer Details, Bank Accounts, etc.)
